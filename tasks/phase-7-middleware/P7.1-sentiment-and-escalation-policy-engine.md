Build the middleware that combines manipulation, sentiment, and business policy signals into consistent escalation behavior.

## Deliverables

- sentiment interface and fallback heuristic mode

- policy hooks for profanity, false promises, and repeated denials

- [ ] escalation decisions are machine-readable and testable

- [ ] the middleware can run without telephony provider code
